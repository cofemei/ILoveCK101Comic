我愛卡提諾漫畫
===============

我愛卡提諾漫畫。

不用慢慢一頁一頁向下換現在可以一次抓下來。

Windows, Linux, Mac

如何使用
===========

```bash
$ git clone https://github.com/cofemei/ILoveCK101Comic.git
$ cd ILoveCK101Comic
$ groovy main.groovy http://m.comic.ck101.com/comic/587

```

你可以在你的桌面發現 `ILoveCk101Comic` 的目錄, Linux 會放在 ~/Desktop/ILoveCk101Comic
